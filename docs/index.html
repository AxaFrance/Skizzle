Télécharger Skizzle

Outil de collaboration entre développeurs

Skizzle permet de récupérer toutes les pull requests des projets qui vous intéressent depuis votre compte Azure DevOps. Dans une même interface, vous voyez d'un coup d'oeil la progression de vos équipes de développement.

Multi équipes

Que vous fassiez partie d'une ou plusieurs équipes, Skizzle est l'outil idéal pour gérer vos revue de code . Surveillez facilement les pull requests de plusieurs projets sur différentes équipes.

Abonnez-vous aux projets qui vous intéressent

Tous les projets de votre compte ne sont pas toujours intéressants. Skizzle récupère les repositories des projets auxquels vous avez accès sur votre compte Azure Devops. Abonnez-vous à ceux qui vous intéressent, Skizzle vous liste les pulls requests en temps réel.

Soyez toujours au courant

À chaque nouvelle pull request, Skizzle vous envoie une notification . Mais Skizzle sait quand vous ne souhaitez pas être dérangé et ne vous enverra rien si votre ordinateur est en mode "Ne pas déranger".

Une liste efficace

Skizzle permet de filtrer la liste affichée en fonction de tags, ou d'autres critères afin d'avoir une liste de pull requests toujours pertinente pour vous.

Skizzle s'adapte à vos préférences

L'interface de Skizzle est conçue pour s'adapter automatiquement à vos préférences de mode sombre ou de mode "Ne pas déranger". Vous pouvez également personnaliser l'interface pour la rendre plus compacte.
